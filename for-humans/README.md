# For Humans

## Load the charter into your LLM

Pre-formatted versions of the charter sized for specific platforms' personalization and system prompt fields. Pick the file for your platform, follow the paste instructions at the top, and you're done.

| File | Platform | Where to paste |
|------|----------|----------------|
| [chatgpt.md](./chatgpt.md) | ChatGPT | Settings → Personalization → Custom Instructions (two fields) |
| [gemini.md](./gemini.md) | Gemini | Settings → Personal context |
| [generic-short.md](./generic-short.md) | Any | System prompt or first message of any new session |

For the full charter, see [CHARTER.md](../CHARTER.md).

Prompts for asking an LLM to review and critique the charter. Each prompt includes the relevant context and a specific assignment — copy, paste the charter where indicated, and send.

| File | Use when |
